Welcome to Ayo World:

Ayo Os is a mobile operating system developed by Ayo Team. It is based on a modified version of the Linux kernel and other open source software, and is designed primarily for touchscreen mobile devices such as smartphones and tablets. In addition, Ayo Team has further developed Ayo Wear OS for wrist watches, each with a specialized user interface. Ayo os was unveiled in 2017, with the first commercial Ayo device launched in September 2019. The operating system has since gone through multiple major releases, with the current version being Ayo ST, released in August 2018. Ayo team released the first Ayo Pro beta on all Ayo Plus1 phones on March 13, 2019. The core Ayo source code is known as Ayo Open Source Project (AOSP), and is primarily licensed under the Apache License.
